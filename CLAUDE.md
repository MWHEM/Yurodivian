# The Yurodivian Site

## Tech stack
- Plain HTML + CSS (no build tool)
- Open `index.html` directly in browser to preview

## Key files
- `index.html` — all page structure and content
- `styles.css` — all styling (colors, layout, typography)
- `assets/` — images (logo, book covers, etc.)

## Design tokens
- Background: `#080d2a`
- Nav background: `#0a1030`
- Gold (headings, accents): `#c8b87a`
- Blue (subtitles, links): `#7ab0d4`
- Body text: `#c8d4e8`
- Card background: `#0f1840`

## Editing rules
- Only edit `index.html` and `styles.css` — do not create extra files unless asked
- Keep all styles in `styles.css`, no inline styles
- Preserve the dark navy + gold color scheme unless explicitly told to change it
